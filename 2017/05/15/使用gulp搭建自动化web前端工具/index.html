为何要使用gulp？

通过代码优于配置的策略，Gulp 让简单的任务简单，复杂的任务可管理；利用 Node.js 流的威力，你可以快速构建项目并减少频繁的 IO 操作；Gulp 严格的插件指南确保插件如你期望的那样简洁高质得工作；通过最少的 API，掌握 Gulp 毫不费力，构建工作尽在掌握：如同一系列流管道。
